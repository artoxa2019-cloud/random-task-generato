# Random Task Generator

**Автор:** Иванов Иван Иванович  ← напишите свои ФИО

## Описание проекта
GUI-приложение на Python (библиотека Tkinter) для генерации случайных задач.  
Позволяет:
- Генерировать случайную задачу из предопределённого списка.
- Добавлять новые задачи с указанием типа (учёба, спорт, работа).
- Фильтровать историю сгенерированных задач по типу.
- Сохранять историю в JSON-файл и загружать её при перезапуске.
- Проверять, что название новой задачи не пустое.

## Требования к системе
- Python 3.6 или новее
- Стандартные библиотеки (tkinter, random, json, os)
