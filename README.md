# Random Task Generator

**Автор:** IШаповалов Никита Витальевич

## Описание
Приложение для генерации случайных задач с графическим интерфейсом (Tkinter).  
Позволяет:
- Генерировать задачу из предустановленного списка.
- Добавлять новые задачи с выбором типа (учёба, спорт, работа).
- Просматривать историю сгенерированных задач.
- Фильтровать историю по типу задачи.
- Автоматически сохранять историю в JSON-файл и загружать при запуске.

## Установка и запуск
1. Установите Python 3.6+.
2. Скачайте файл `main.py`.
3. Запустите в терминале:
   ```bash
   python main.py
